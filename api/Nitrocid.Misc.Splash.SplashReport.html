    ReportProgress(string, int, bool, Exception?, SplashReportSeverity, ISplash?, params object[])

    Reports the progress for the splash screen while the kernel is booting.

    Declaration
    public static void ReportProgress(string Text, int Progress, bool force = false, Exception? exception = null, SplashReportSeverity severity = SplashReportSeverity.Info, ISplash? splash = null, params object[] Vars)
    Parameters
    Type Name Description
    string Text

    The progress text to indicate how did the kernel progress
    int Progress

    The progress percentage from 0 to 100 to increment to the overall percentage
    bool force

    Force report progress to splash
    Exception exception

    Exception to use for warnings and errors
    SplashReportSeverity severity

    The severity of the report
    ISplash splash

    Splash interface. Use null to use default splash.
    object[] Vars

    Variables to be expanded to text
    Remarks

    If the kernel has booted successfully, it will act like the normal printing command. If this routine was called during boot,
    it will report the progress to the splash system. You can force it to report the progress by passing force.
