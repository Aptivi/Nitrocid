    GetDateFromCalendarNoCulture(CalendarTypes)

    Gets a date from a specified calendar without accessing the culture information, depending on the implementation of the calendar

    Declaration
    public static (int year, int month, int day, Calendar calendarInfo) GetDateFromCalendarNoCulture(CalendarTypes calendar)
    Parameters
    Type Name Description
    CalendarTypes calendar

    Calendar to use during conversion
    Returns
    Type Description
    (int year, int month, int day, Calendar calendarInfo)

    Resulting year, month, day, and a calendar information instance
    Remarks

    Nitrocid calendars and all calendars that don't override the base Calendar usually access the culture information to get the calendar. You can override the Calendar property so that it doesn't access the culture information to make this function behave differently from the GetDateFromCalendar(CalendarTypes) and the GetDateFromCalendar(DateTime, CalendarTypes) functions that access the culture information.
