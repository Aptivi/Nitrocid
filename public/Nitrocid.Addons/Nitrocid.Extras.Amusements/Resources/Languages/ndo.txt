Rilis pertama kalinya
Kernel ini pertama kali dirilis pada tanggal 22 Februari 2018 untuk membuat simulator minimalis yang benar-benar mensimulasikan cara kerja inti kernel.
Dulu ia hanya menampung satu shell dan beberapa perintah, dan hanya berfungsi pada Windows karena menggunakan alat Windows Management Instrumentation.
Ia juga tidak memiliki dokumentasinya sendiri.
Penyempurnaan Kernel
Versi kernel dirilis, seperti rilis utama kedua untuk menambahkan pewarnaan dan fitur dasar, dan rilis utama keempat untuk menambahkan konfigurasi.
Dokumentasi dimulai pada rilis utama keenam, dan berbagai hal bagus telah ditambahkan perlahan ke kernel.
Versi saat ini
Versi ini sekarang menyempurnakan kernel hingga tidak lagi berperilaku seperti versi lama.
Karena fitur-fitur inovatif baru telah dirilis, kami memutuskan untuk memberi nama kernel ini...
